# Claude Code Başlangıç Eğitimi

Claude Code'u sıfırdan öğreten, Türkçe hazırlanmış 8 modüllük pratik eğitim.

---

## Başlamadan Önce

### 1. Node.js kur (kurulu değilse)

```bash
node --version
```

Versiyon numarası görüyorsan kurulu demektir. Görmüyorsan [nodejs.org](https://nodejs.org) adresinden LTS sürümünü indir ve kur.

### 2. Claude Code'u kur

```bash
npm install -g @anthropic-ai/claude-code
```

Kurulumu doğrula:

```bash
claude --version
```

Bir versiyon numarası görüyorsan (örn. `2.1.196`) hazırsın.

### 3. Bu repoyu bilgisayarına indir

```bash
git clone https://github.com/emrechef/claude-code-baslangic.git
cd claude-code-baslangic
```

### 4. Eğitime başla

```bash
# İlk olarak bu dosyayı oku
open BASLA-BURADAN.md
```

Veya dosya yöneticisinden `BASLA-BURADAN.md` dosyasını aç.
